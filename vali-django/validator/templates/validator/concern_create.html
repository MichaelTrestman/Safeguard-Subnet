What is a concern?

A concern is a first-person worry about AI BEHAVIOR, written from the operator's perspective. The subject should always be the AI, and the verb should describe the bad thing happening:

  • YES "I'm concerned the AI will leak credentials"
  • YES "I'm concerned the AI will walk a rushed user through staging their .env file into a commit"

NOT user prompts, NOT attack patterns, NOT detection rules:

  • NO "Prompts asking the assistant to write malware"
  • NO "Jailbreak attempts using DAN roleplay"

After creating the concern, you'll add:

  • Detection cues — what the AI's bad response looks like (substring / regex / semantic signatures)
  • User triggers — what kinds of user prompts might elicit it (prompt / persona / context framings)
